البدء مع Rust

ثبّت Rust، اكتب أول برنامج لك، وتعلّم أساسيات Cargo, البوابة إلى برمجة الأنظمة.

1

ما هي Rust ولماذا نستخدمها؟

Rust هي لغة برمجة أنظمة تعمل بسرعة فائقة، تمنع أخطاء الـ Memory، وتضمن Thread Safety. صدرت في عام 2010 ويتم تطويرها باستمرار من قبل مجتمع مفتوح المصدر نشط. تجمع Rust بين أداء لغات مثل C وC++ مع ضمانات Memory Safety التي تمنع فئات كاملة من الأخطاء في Compile-time.

الركائز الثلاث لـ Rust

  • Memory Safety: تمنع Rust إلغاء Reference Null Pointers، وBuffer Overflow، وأخطاء الاستخدام بعد التحرير من خلال نظام Ownership. لا يوجد Garbage Collector, Rust Compiler يضمن إدارة الـ Memory بشكل صحيح.
  • الأداء: مع Zero-cost Abstractions ودون أي عبء إضافي أثناء التشغيل، تعمل كود Rust بنفس سرعة كود C المكافئة. تحصل على ميزات لغة عالية المستوى دون التضحية بالسرعة.
  • التزامن: يزيل نظام Ownership في Rust Data Races في Compile-time. اكتب كود متزامنة مضمونة Thread Safety دون الاعتماد على Locks.

هذه الخصائص تجعل Rust مثالية لـ:

  • برمجة الأنظمة (أنظمة التشغيل، الأنظمة المضمنة)
  • خوادم الويب والتطبيقات الشبكية
  • التطبيقات الحساسة للأداء
  • WebAssembly والتطبيقات المبنية على المتصفح
  • مشاريع البلوكتشين والعملات الرقمية
  • محركات الألعاب وتطبيقات الرسومات
2

تثبيت Rust

أسهل طريقة لتثبيت Rust هي باستخدام rustup، الذي يدير إصدارات Rust والأدوات المرتبطة بها. إليك كيفية البدء:

خطوات التثبيت

Shell 
# On macOS, Linux, or Windows (with WSL/Git Bash):
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
# This installs:
# - rustc: The Rust compiler
# - cargo: The package manager and build system
# - rustup: The installer and version manager

بعد التثبيت، تحقق من أن كل شيء يعمل بشكل صحيح:

Shell 
rustc --version
cargo --version

يجب أن ترى معلومات الإصدار لكلتا الأداتين. إذا كنت على Windows، يمكنك أيضاً تنزيل وتشغيل المثبّت من https://rustup.rs/.

💡

حافظ على تحديث Rust

يتم تحديث Rust بشكل متكرر. لتحديث تثبيت Rust لديك، قم ببساطة بتشغيل rustup update من الطرفية.

أول برنامج Rust: Hello World

لنكتب أول برنامج Rust لك. أنشئ مجلداً جديداً وملفاً باسم main.rs:

Rust 
fn main() {
    println!("Hello, world!");
}

الآن قم بترجمته وتشغيله:

Shell 
rustc main.rs
./main

يجب أن ترى Hello, world! مطبوعاً في الطرفية. تهانينا! لقد كتبت أول برنامج Rust لك.

✨

البرنامج الأول اكتمل!

لقد قمت بترجمة وتنفيذ أول برنامج Rust لك. الكلمة المفتاحية fn تعلن عن Function، وprintln! هو ماكرو يطبع النص في وحدة التحكم.

4

Cargo: مدير الحزم ونظام البناء في Rust

لأي شيء يتجاوز البرامج البسيطة، ستحتاج لاستخدام Cargo. هو مدير الحزم ونظام البناء الرسمي في Rust الذي يتعامل مع Dependencies وCompilation وTesting وDocumentation.

إنشاء مشروع جديد

أنشئ مشروع Cargo جديداً بأمر واحد:

Shell 
cargo new hello_world
cd hello_world

هذا ينشئ هيكل المجلدات التالي:

Text 
hello_world/
├── Cargo.toml          # Project manifest with metadata and dependencies
└── src/
    └── main.rs         # Your source code

ملف Cargo.toml

يحتوي هذا الملف على بيانات المشروع الوصفية وDependencies. إليك مثالاً نموذجياً:

TOML 
[package]
name = "hello_world"
version = "0.1.0"
edition = "2021"
[dependencies]
# Add dependencies here
serde = "1.0"
tokio = { version = "1.0", features = ["full"] }

أوامر Cargo

أوامر Cargo الرئيسية التي ستستخدمها يومياً:

  • cargo build: يترجم مشروعك في وضع التصحيح. ينشئ ملفاً تنفيذياً في target/debug/.
  • cargo run: يترجم ويشغّل مشروعك فوراً. مفيد أثناء التطوير.
  • cargo build --release: يترجم مع تفعيل التحسينات. ينشئ ملفاً تنفيذياً في target/release/. أبطأ في Compilation لكن أسرع بكثير في التشغيل.
  • cargo test: يشغّل جميع Testingات في مشروعك.
  • cargo check: يتحقق بسرعة من أن شيفرتك تُترجم دون إنشاء ملف تنفيذي. ممتاز للحصول على ملاحظات سريعة أثناء التطوير.
  • cargo fmt: ينسّق شيفرتك تلقائياً وفقاً لإرشادات أسلوب Rust.
  • cargo clippy: أداة تحليل تقدم اقتراحات لتحسين شيفرتك.
  • cargo doc --open: ينشئ ويفتح توثيق HTML لمشروعك وتبعياته.
5

بناء وتشغيل مشروع Cargo

Shell 
# Create the project
cargo new my_app
# Navigate to the project
cd my_app
# Edit src/main.rs with your code
# Check if it compiles (fast, no binary)
cargo check
# Run during development (slower compile, good for debugging)
cargo run
# Build for production (much faster execution)
cargo build --release
# Run the release binary directly
./target/release/my_app
💡

نصيحة احترافية

استخدم cargo check أثناء التطوير للحصول على ملاحظات سريعة. استخدم cargo build --release فقط عندما تكون جاهزاً للنشر أو اختبار أداء تطبيقك.

6

أول مشروع Cargo حقيقي لك

لننشئ مشروع Cargo أكثر إثارة يوضح المفاهيم الأساسية:

Rust 
// File: src/main.rs
use std::io;
fn main() {
    let mut input = String::new();
    println!("Welcome to the Rust Guessing Game!");
    println!("Enter a number between 1 and 100:");
    io::stdin()
        .read_line(&mut input)
        .expect("Failed to read line");
    let number: u32 = input
        .trim()
        .parse()
        .expect("Please enter a valid number");
    println!("You entered: {}", number);
    if number > 50 {
        println!("That's a high number!");
    } else {
        println!("That's a low number!");
    }
}

يوضح هذا البرنامج عدة ميزات أساسية في Rust:

  • استخدام وحدة std::io من مكتبة Rust القياسية
  • Variables الـ Mutable باستخدام let mut
  • تحليل النصوص وError Handling باستخدام expect()
  • Control Flow باستخدام if/else

قم ببنائه وتشغيله:

Shell 
cargo run
