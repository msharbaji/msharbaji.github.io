أساسيات Rust

Variables و Data Types و Functions و Control Flow, اللبنات الأساسية لكل برنامج Rust.

1

Variables و Mutability

في Rust، Variables Immutable افتراضياً. هذه ميزة أمان أساسية تمنع التعديلات العرضية وتجعل الكود أسهل في الفهم. لجعل Variable Mutable، تستخدم الكلمة المفتاحية mut بشكل صريح.

Variables Immutable

Rust 
fn main() {
    let x = 5;
    println!("x = {}", x);
    // This will cause a compile error:
    // x = 6;
}

Variables الـ Mutable

عندما تحتاج لتغيير قيمة Variable، أعلن عنه كـ Mutable بشكل صريح:

Rust 
fn main() {
    let mut x = 5;
    println!("x = {}", x);
    x = 6;
    println!("x = {}", x);
}

Constants

الـ Constants دائماً Immutable ويجب تحديد نوعها. يمكن الإعلان عنها على مستوى الوحدة:

Rust 
const MAX_POINTS: u32 = 100_000;
const PI: f64 = 3.14159;
fn main() {
    println!("Maximum points: {}", MAX_POINTS);
    println!("Pi: {}", PI);
}
💡

Constants مقابل Variables

لا يمكن عمل Shadowing لـ Constants، ويتم تقييمها في Compile-time. يمكن تظليل Variables (إعادة الإعلان عنها بنفس الاسم)، مما يسمح بتغيير القيمة وربما نوعها.

Shadowing

يسمح لك الـ Shadowing بإعادة استخدام أسماء Variables وحتى تغيير أنواعها:

Rust 
fn main() {
    let x = 5;
    let x = x + 1;  // x is now 6
    let x = x * 2;  // x is now 12
    // You can even change the type:
    let spaces = "   ";
    let spaces = spaces.len();  // spaces is now a number (3)
    println!("x = {}, spaces = {}", x, spaces);
}

Data Types

Rust لغة ذات أنواع ثابتة، مما يعني أن الـ Compiler يجب أن يعرف نوع كل Variable في Compile-time. غالباً ما يستطيع Rust استنتاج الأنواع، لكن يمكنك أيضاً تحديدها بشكل صريح.

Scalar Types

تمثل Scalar Types قيماً مفردة. لدى Rust أربعة أنواع قياسية أساسية.

أنواع Integers

أعداد صحيحة بإشارة وبدون إشارة بأحجام مختلفة:

Rust 
fn main() {
    // Signed integers (can be negative)
    let a: i32 = -42;     // 32-bit signed
    let b: i64 = -1000;   // 64-bit signed
    let c: i8 = -5;       // 8-bit signed (range -128 to 127)
    // Unsigned integers (always positive)
    let d: u32 = 42;      // 32-bit unsigned
    let e: u64 = 1000;    // 64-bit unsigned
    let f: u8 = 255;      // 8-bit unsigned (range 0 to 255)
    // Without annotation, integers default to i32
    let g = 100;
    println!("a={}, b={}, c={}, d={}, e={}, f={}, g={}", a, b, c, d, e, f, g);
}

أنواع الأعداد العشرية

أعداد عشرية وفق معيار IEEE 754:

Rust 
fn main() {
    let x: f64 = 3.14159;    // 64-bit float (default)
    let y: f32 = 2.71828;    // 32-bit float
    // Operations on floats
    let sum = x + y as f64;
    let product = x * y as f64;
    println!("sum = {}, product = {}", sum, product);
}

النوع المنطقي

قيم منطقية صح/خطأ:

Rust 
fn main() {
    let is_active: bool = true;
    let is_ready = false;  // Type inferred as bool
    // Booleans are used in conditionals
    if is_active && !is_ready {
        println!("Active but not ready");
    }
}

نوع الحرف

قيم Unicode العددية:

Rust 
fn main() {
    let c: char = 'z';
    let emoji: char = '🦀';
    let greek: char = 'π';
    println!("char: {}, emoji: {}, greek: {}", c, emoji, greek);
}

Compound Types

تجمع Compound Types عدة قيم في نوع واحد. النوعان المركبان الأساسيان هما Tuples و Arrays.

Tuples

مجموعات ذات طول ثابت من قيم بأنواع مختلفة محتملة:

Rust 
fn main() {
    // Tuple with mixed types
    let tup: (i32, f64, u8) = (500, 6.4, 1);
    // Access tuple elements with destructuring
    let (x, y, z) = tup;
    println!("x = {}, y = {}, z = {}", x, y, z);
    // Access tuple elements with dot notation
    println!("First element: {}", tup.0);
    println!("Second element: {}", tup.1);
    // Empty tuple (unit type)
    let unit: () = ();
}

Arrays

مجموعات ذات طول ثابت من قيم من نفس النوع:

Rust 
fn main() {
    // Array with explicit type and length
    let a: [i32; 5] = [1, 2, 3, 4, 5];
    // Array with repeated value
    let b = [3; 5];  // [3, 3, 3, 3, 3]
    // Access array elements
    println!("First element: {}", a[0]);
    println!("Last element: {}", a[4]);
    // Iterate over array
    for element in a.iter() {
        println!("Element: {}", element);
    }
    // Get array length
    println!("Array length: {}", a.len());
}

Functions

Functions هي اللبنات الأساسية في Rust. يتم الإعلان عنها باستخدام الكلمة المفتاحية fn ويمكن أن تأخذ معاملات وتعيد قيماً.

صيغة الـ Function الأساسية

Rust 
fn add(a: i32, b: i32) -> i32 {
    a + b
fn main() {
    let result = add(5, 3);
    println!("5 + 3 = {}", result);
}

القيم المعادة والـ Expressions

في Rust، يتم إعادة آخر Expression في الـ Function تلقائياً. لاحظ عدم وجود فاصلة منقوطة في الـ Expression الأخير:

Rust 
fn multiply(a: i32, b: i32) -> i32 {
    a * b  // No semicolon - this is a return statement
fn add_one(x: i32) -> i32 {
    x + 1  // This will be returned
fn main() {
    let result1 = multiply(4, 5);
    let result2 = add_one(10);
    println!("4 * 5 = {}", result1);
    println!("10 + 1 = {}", result2);
}
⚠️

انتبه للفواصل المنقوطة!

إضافة فاصلة منقوطة بعد الـ Expression الأخير يحوله إلى Statement، مما يعني أن الـ Function تعيد نوع الوحدة () بدلاً من القيمة المتوقعة.

Functions ذات المعاملات المتعددة

Rust 
fn calculate_rectangle_area(width: f64, height: f64) -> f64 {
    width * height
fn print_coordinates(x: i32, y: i32) {
    println!("Coordinates: ({}, {})", x, y);
fn main() {
    let area = calculate_rectangle_area(15.5, 8.0);
    println!("Area: {}", area);
    print_coordinates(10, 20);
}

Control Flow

توفر Rust عدة طرق للتحكم بتدفق برنامجك: الشرطيات (if/else) والـ Loops (loop، while، for).

Expressions if و else

Rust 
fn check_number(n: i32) {
    if n > 0 {
        println!("{} is positive", n);
    } else if n < 0 {
        println!("{} is negative", n);
    } else {
        println!("{} is zero", n);
    }
fn main() {
    check_number(5);
    check_number(-3);
    check_number(0);
}

استخدام if كـ Expression: بما أن if هو Expression في Rust، يمكنك استخدامه لتعيين قيم:

Rust 
fn main() {
    let condition = true;
    let number = if condition { 5 } else { 6 };
    println!("The value is: {}", number);  // Prints: The value is: 5
    // Both branches must return the same type
    let age = 25;
    let category = if age >= 18 { "adult" } else { "minor" };
    println!("Category: {}", category);
}

Statement loop

الكلمة المفتاحية loop تنشئ Loop لا نهائية تعمل حتى تخرج منها بشكل صريح:

Rust 
fn main() {
    let mut counter = 0;
    loop {
        println!("Counter: {}", counter);
        counter += 1;
        if counter == 3 {
            break;  // Exit the loop
        }
    }
    println!("Loop finished");
}

إعادة قيم من الـ Loops: يمكنك إعادة قيمة من Loop باستخدام break مع Expression:

Rust 
fn main() {
    let mut counter = 0;
    let result = loop {
        counter += 1;
        if counter == 5 {
            break counter * 2;  // Break with a value
        }
    };
    println!("Result: {}", result);  // Prints: Result: 10
}

Loop while

Rust 
fn main() {
    let mut number = 3;
    while number != 0 {
        println!("{}!", number);
        number -= 1;
    }
    println!("LIFTOFF!");
}

Loop for

Loop for هي الطريقة الأكثر اصطلاحية للتكرار في Rust:

Rust 
fn main() {
    // Iterate over a range
    for i in 1..4 {
        println!("{}!", i);
    }
    // Iterate with inclusive range (includes 5)
    for i in 1..=5 {
        println!("Number: {}", i);
    }
    // Iterate over array
    let arr = [10, 20, 30, 40];
    for element in arr.iter() {
        println!("Element: {}", element);
    }
    // Reverse iteration
    for i in (1..=3).rev() {
        println!("{}!", i);
    }
}
