الماكرو

Metaprogramming في Rust, Declarative Macros وProcedural Macros وDerive Macros.

1

Declarative Macros (macro_rules!)

يتيح Declarative Macros Pattern Matching على تسلسلات الرموز وتوليد الكود. وهو أكثر أنواع الماكرو شيوعاً في Rust ويستخدم صيغة macro_rules!.

Rust
// Simple declarative macro
macro_rules! say_hello {
    () => {
        println!("Hello!");
    };
// Macro with arguments
macro_rules! create_function {
    ($func_name:ident) => {
        fn $func_name() {
            println!("You called {:?}", stringify!($func_name));
        }
    };
fn main() {
    say_hello!();
    create_function!(my_function);
    my_function();
}
Rust
// Variadic macro with repetition
macro_rules! vec_repeat {
    ($item:expr; $count:expr) => {
        {
            let mut v = Vec::new();
            for _ in 0..$count {
                v.push($item);
            }
            v
        }
    };
// Macro with multiple arguments
macro_rules! calculate {
    ($a:expr, +, $b:expr) => { $a + $b };
    ($a:expr, -, $b:expr) => { $a - $b };
    ($a:expr, *, $b:expr) => { $a * $b };
fn main() {
    let v = vec_repeat!(5; 3);
    println!("Vector: {:?}", v);
    println!("10 + 5 = {}", calculate!(10, +, 5));
    println!("10 - 5 = {}", calculate!(10, -, 5));
    println!("10 * 5 = {}", calculate!(10, *, 5));
}
💡

صيغة استدعاء الماكرو

يمكن استدعاء الماكرو بعلامة ! مع أقواس عادية () أو أقواس مربعة [] أو أقواس معقوفة {}. الاختيار أسلوبي, فمثلاً vec![] يستخدم الأقواس المربعة، وprintln!() يستخدم الأقواس العادية.

2

Patterns التكرار ($()*)

تتيح Patterns التكرار للماكرو التعامل مع عدد Variable من المعاملات. صيغة $(...)* تطابق صفراً أو أكثر من التكرارات، بينما $(...)+ تطابق واحداً أو أكثر.

Rust
// Macro for printing multiple values
macro_rules! print_vals {
    ($($val:expr),*) => {
        $(
            println!("{}", $val);
        )*
    };
// Macro for creating tuples
macro_rules! make_tuple {
    ($($item:expr),*) => {
        ($($item),*)
    };
fn main() {
    print_vals!("Hello", 42, 3.14, true);
    let tuple = make_tuple!(1, 2, "three", 4.0);
    println!("Tuple: {:?}", tuple);
}
Rust
// HashMap creation macro
macro_rules! map {
    ($($key:expr => $value:expr),* $(,)?) => {
        {
            let mut m = std::collections::HashMap::new();
            $(
                m.insert($key, $value);
            )*
            m
        }
    };
fn main() {
    let m = map!{
        "name" => "Alice",
        "age" => "30",
        "city" => "Boston",
    };
    for (k, v) in &m {
        println!("{}: {}", k, v);
    }
}
✨

دعم الفاصلة اللاحقة

نمط $(,)? في نهاية ذراع الماكرو يسمح بفاصلة لاحقة اختيارية، مما يجعل استخدام الماكرو أكثر سلاسة.

3

Patterns الماكرو الشائعة

تظهر عدة Patterns بشكل متكرر في كود Rust. تشمل طباعة التنقيح والتأكيدات المخصصة وتوليد الكود بنمط البناء.

Rust
// Debug printing macro
macro_rules! dbg_macro {
    ($val:expr) => {
        match &$val {
            tmp => {
                eprintln!("[{}:{}] {} = {:#?}",
                    file!(),
                    line!(),
                    stringify!($val),
                    &tmp);
                tmp
            }
        }
    };
// Assert with custom message
macro_rules! assert_msg {
    ($cond:expr, $msg:expr) => {
        assert!($cond, "Custom message: {}", $msg);
    };
fn main() {
    let x = 42;
    let _ = dbg_macro!(x);
    assert_msg!(x > 0, "x should be positive");
    println!("Assertions passed");
}
Rust
// Builder pattern macro
macro_rules! builder {
    ($name:ident { $($field:ident: $typ:ty),* $(,)? }) => {
        struct $name {
            $(
                $field: $typ,
            )*
        }
        impl $name {
            fn new() -> Self {
                Self {
                    $(
                        $field: Default::default(),
                    )*
                }
            }
        }
    };
builder!(Config {
    host: String,
    port: u16,
    debug: bool,
});
fn main() {
    let _config = Config::new();
    println!("Config created");
}
💡

ماكرو الأدوات المدمجة

يوفر Rust ماكرو file!() وline!() وcolumn!() وstringify!() كماكرو مدمجة, وهي مثالية لإنشاء أدوات التنقيح والتسجيل الخاصة بك.

Procedural Macros

يعمل Procedural Macros على تدفقات الرموز ويمنح قوة أكبر من Declarative Macros. يُعرَّف في حزمة منفصلة مع تعيين proc-macro = true في Cargo.toml.

Rust
// A procedural macro using the proc_macro crate
// This would be in a separate crate with proc-macro = true
extern crate proc_macro;
use proc_macro::TokenStream;
use quote::quote;
use syn;
#[proc_macro]
pub fn my_macro(input: TokenStream) -> TokenStream {
    let input = syn::parse_macro_input!(input as syn::LitInt);
    let n = input.base10_parse::<i32>().unwrap();
    let expanded = quote! {
        {
            let value = #n;
            println!("Value: {}", value);
            value
        }
    };
    TokenStream::from(expanded)
}
⚠️

متطلب الحزمة المنفصلة

يجب أن يكون Procedural Macros في حزمته الخاصة مع تعيين proc-macro = true في Cargo.toml. يعتمد على حزمة syn للتحليل وحزمة quote لتوليد الرموز.

5

Derive Macros

يقوم Derive Macros بتنفيذ الـ Traits تلقائياً لـ Structs وEnums. الاشتقاقات القياسية مثل Debug وClone وPartialEq مدمجة، ويمكنك إنشاء اشتقاقات مخصصة.

Rust
use std::fmt;
// Standard derive macros
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
struct Point {
    x: i32,
    y: i32,
// Custom trait that could be derived with a proc macro
trait Drawable {
    fn draw(&self);
impl Drawable for Point {
    fn draw(&self) {
        println!("Drawing point at ({}, {})", self.x, self.y);
    }
fn main() {
    let p = Point { x: 10, y: 20 };
    println!("{:?}", p);
    p.draw();
}
6

Attribute Macros

يُعلَّق Attribute Macros على العناصر ويحوّلها. كثير منها مدمج في Rust ويُستخدم يومياً, مثل #[test] و#[cfg] و#[deprecated] و#[inline] وهي جميعها ماكرو سمات.

Rust
// #[test] - marks functions as tests
#[cfg(test)]
mod tests {
    #[test]
    fn test_example() {
        assert_eq!(2 + 2, 4);
    }
// #[deprecated] - marks items as deprecated
#[deprecated(since = "0.2.0", note = "use new_function instead")]
fn old_function() {
    println!("This is old");
// #[inline] - hints to compiler to inline function
#[inline]
fn simple_operation(x: i32) -> i32 {
    x * 2
fn main() {
    let _result = simple_operation(5);
}
✨

Attribute Macros المخصصة

تستخدم أطر العمل مثل Actix Web وRocket Attribute Macros المخصصة بكثرة, فمثلاً #[get("/")] و#[tokio::main] و#[actix_web::main] هي جميعها ماكرو سمات إجرائية.

متى تستخدم الماكرو بدلاً من Functions

يجب أن تكون Functions خيارك الافتراضي, فهي أبسط وأوضح وأسهل في التنقيح. استخدم الماكرو عندما تحتاج إمكانيات لا توفرها Functions.

Rust
// Use functions when possible - they're simpler and clearer
fn add(a: i32, b: i32) -> i32 {
    a + b
// Use macros for:
// 1. Variable argument counts
macro_rules! sum {
    ($($num:expr),+) => {
        0 $( + $num )*
    };
// 2. Code generation patterns
macro_rules! implement_ops {
    ($($op_name:ident -> $op_fn:expr)*) => {
        $(
            fn $op_name(a: i32, b: i32) -> i32 {
                $op_fn(a, b)
            }
        )*
    };
implement_ops! {
    multiply -> |a, b| a * b
    divide -> |a, b| if b != 0 { a / b } else { 0 }
fn main() {
    println!("Sum: {}", sum!(1, 2, 3, 4, 5));
    println!("Multiply: {}", multiply(3, 4));
}
💡

دليل الاختيار

استخدم Functions للمنطق العادي. استخدم Declarative Macros للمعاملات الـ Variableة والـ Patterns البسيطة. استخدم Procedural Macros لتوليد الكود المعقد والاشتقاقات المخصصة. فضّل دائماً الأداة الأبسط التي تنجز المهمة.
