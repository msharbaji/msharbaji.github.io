Structs و Enums

عرّف أنواعاً مخصصة باستخدام struct، ونمذج Variables باستخدام enum، واتقن Pattern Matching في Rust.

1

تعريف واستخدام Structs

الـ struct هو نوع بيانات مخصص يجمع عدة قيم مرتبطة معاً. هناك ثلاثة أنواع من Structs في Rust.

تعريف الهيكل

Rust
struct User {
    username: String,
    email: String,
    sign_in_count: u64,
    active: bool,
fn main() {
    let mut user = User {
        username: String::from("john_doe"),
        email: String::from("john@example.com"),
        sign_in_count: 1,
        active: true,
    };
    user.sign_in_count += 1;
    println!("User: {} ({})", user.username, user.email);
    println!("Sign in count: {}", user.sign_in_count);
}

اختصار تهيئة الحقول

عندما تتطابق أسماء Variables مع أسماء الحقول، يمكنك استخدام الصيغة المختصرة:

Rust
fn create_user(username: String, email: String) -> User {
    User {
        username,     // Same as username: username
        email,        // Same as email: email
        sign_in_count: 1,
        active: true,
    }
fn main() {
    let user = create_user(
        String::from("jane_doe"),
        String::from("jane@example.com"),
    );
}

صيغة تحديث الهيكل

أنشئ هيكلاً جديداً من هيكل موجود مع تغيير بعض الحقول فقط:

Rust
fn main() {
    let user1 = User {
        username: String::from("john"),
        email: String::from("john@example.com"),
        sign_in_count: 1,
        active: true,
    };
    let user2 = User {
        username: String::from("jane"),
        ..user1  // Use remaining fields from user1
    };
}

Tuple Structs

هياكل بدون حقول مسماة:

Rust
struct Color(i32, i32, i32);
struct Point(i32, i32, i32);
fn main() {
    let black = Color(0, 0, 0);
    let origin = Point(0, 0, 0);
    println!("Color red: {}", black.0);
    println!("Point x: {}", origin.0);
}

Structs الشبيهة بالوحدة

هياكل بدون أي حقول:

Rust
struct AlwaysEqual;
fn main() {
    let subject = AlwaysEqual;
}
💡

نصيحة

استخدم صيغة تحديث الهيكل (..other_struct) لإنشاء نسخ جديدة تتشارك معظم الحقول مع هيكل موجود. هذا يحافظ على الكود نظيفاً وقابلاً للقراءة.

2

صيغة الـ Methods

الـ Methods هي دوال تُعرَّف ضمن سياق struct (أو enum أو trait). تأخذ self كأول معامل لها.

تعريف Functions باستخدام impl

Rust
struct Rectangle {
    width: u32,
    height: u32,
impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }
    fn can_hold(&self, other: &Rectangle) -> bool {
        self.width > other.width && self.height > other.height
    }
    fn set_width(&mut self, width: u32) {
        self.width = width;
    }
fn main() {
    let mut rect1 = Rectangle { width: 30, height: 50 };
    let rect2 = Rectangle { width: 20, height: 40 };
    println!("Area: {}", rect1.area());
    println!("Can hold: {}", rect1.can_hold(&rect2));
    rect1.set_width(35);
}

الـ Methods بدون self

الـ Methods التي لا تأخذ self تُستدعى باستخدام صيغة :::

Rust
impl Rectangle {
    fn square(size: u32) -> Rectangle {
        Rectangle { width: size, height: size }
    }
fn main() {
    let sq = Rectangle::square(10);
    println!("Square area: {}", sq.area());
}
✨

ملاحظة مهمة

الـ Methods (مثل Rectangle::square) تعمل كمُنشئات. تُستدعى باستخدام :: بدلاً من . لأنها لا تعمل على نسخة محددة.

3

Enums

تتيح لك Enums تعريف نوع بمجموعة من Variables الممكنة. يمكن أن يحمل كل Variable بيانات مختلفة مرتبطة به.

تعداد بسيط

Rust
enum Direction {
    North,
    South,
    East,
    West,
fn main() {
    let direction = Direction::North;
}

Enums مع بيانات مرتبطة

Rust
enum IpAddr {
    V4(String),
    V6(String),
enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(i32, i32, i32),
fn main() {
    let ip1 = IpAddr::V4(String::from("127.0.0.1"));
    let ip2 = IpAddr::V6(String::from("::1"));
    let msg1 = Message::Quit;
    let msg2 = Message::Move { x: 10, y: 20 };
    let msg3 = Message::Write(String::from("hello"));
    let msg4 = Message::ChangeColor(255, 0, 0);
}

Functions على Enums

Rust
impl Message {
    fn call(&self) {
        println!("Calling message method");
    }
fn main() {
    let msg = Message::Write(String::from("hello"));
    msg.call();
}
💡

نصيحة

Enums في Rust أقوى بكثير من Enums في معظم اللغات. يمكن أن يحمل كل Variable أنواعاً وكميات مختلفة من البيانات، مما يجعلها مثالية لنمذجة آلات الحالة وMessage Passing.

4

تعداد Option

تعداد Option هو أحد أقوى ميزات Rust. يمثل قيمة اختيارية:

Rust
enum Option<T> {
    Some(T),
    None,
}

يستخدم Rust تعداد Option لتمثيل القيم التي قد لا تكون موجودة. هذا يلغي الحاجة لمؤشرات null:

Rust
fn main() {
    let some_number = Some(5);
    let some_char = Some('e');
    let absent_number: Option<i32> = None;
    // Extract the value with match
    match some_number {
        Some(n) => println!("Number: {}", n),
        None => println!("No number"),
    }
}
✨

ملاحظة مهمة

لا يوجد null في Rust. بدلاً من ذلك، يُجبرك Option<T> على معالجة حالة غياب القيمة صراحةً. لن يسمح لك المُترجم باستخدام Option<T> كما لو كان T, يجب فك تغليفه أولاً.

5

Pattern Matching باستخدام match

يتيح لك Expression match مقارنة قيمة مع عدة Patterns وتنفيذ الكود بناءً على الـ Pattern المطابق:

match بسيط

Rust
enum Coin {
    Penny,
    Nickel,
    Dime,
    Quarter,
fn value_in_cents(coin: Coin) -> u8 {
    match coin {
        Coin::Penny => {
            println!("Lucky penny!");
            1
        }
        Coin::Nickel => 5,
        Coin::Dime => 10,
        Coin::Quarter => 25,
    }
fn main() {
    println!("Penny value: {}", value_in_cents(Coin::Penny));
}

match مع تعدادات تحتوي على بيانات

Rust
enum UsState {
    Alabama,
    California,
enum Coin {
    Penny,
    Nickel,
    Dime,
    Quarter(UsState),
fn value_in_cents(coin: Coin) -> u8 {
    match coin {
        Coin::Quarter(state) => {
            println!("State quarter from {:?}", state);
            25
        }
        _ => 10,  // Catch-all pattern
    }
}
⚠️

تحذير

يجب أن يكون match شاملاً, يجب تغطية كل قيمة ممكنة. استخدم _ كنمط شامل عندما لا تحتاج لمعالجة كل Variable على حدة.

6

صيغة if let

عندما تهتم بنمط واحد فقط، فإن if let أكثر اختصاراً من match:

Rust
fn main() {
    let config_max = Some(3u8);
    // Using match (verbose)
    match config_max {
        Some(max) => println!("Maximum is: {}", max),
        _ => {},
    }
    // Using if let (concise)
    if let Some(max) = config_max {
        println!("Maximum is: {}", max);
    }
}

يمكنك أيضاً استخدام else مع if let:

Rust
fn main() {
    let coin = Coin::Penny;
    if let Coin::Quarter(state) = coin {
        println!("State quarter from {:?}", state);
    } else {
        println!("Not a quarter");
    }
}
💡

نصيحة

استخدم if let عندما تحتاج لمطابقة نمط واحد فقط. استخدم match عندما تحتاج لمعالجة Patterns متعددة أو تريد أن يتحقق المُترجم من الشمولية.
